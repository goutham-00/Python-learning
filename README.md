# Python-learning

My Python learning path using Mimo app. Building foundation for Google/Microsoft interviews.

## Projects Included

### 1. Comparison.py
Compare student data entries. Check for duplicates, same grades, and higher scores.
Skills: Variables, Booleans, Conditional logic

### 2. Sales_tracker.py  
Track sales data for 5 days. Calculate total, average, and best day.
Skills: Math operations, Data analysis, max() function

### 3. Converter.py
Convert KM to Miles. Check if distance is long or short.
Skills: Unit conversion, Conditional checks

## Goal
Learning Python basics, conditionals, and small real-world projects.
